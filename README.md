## 🚀 What is StarNote?

StarNote is a high-performance digital notebook that prioritizes a **subscription-free** experience without compromising on pro-level tools. Built on a proprietary ink engine, it offers:

  * **Ultimate Infinite Canvas:** Limitless vertical and horizontal scroll for expansive brainstorming and deep derivations.
  * **Pro-Level PDF Toolkit:** Advanced annotation, smart inversion for dark mode, and seamless page expansion.
  * **Precision Creation:** Low-latency handwriting, intelligent shape recognition, and recently added **Smart Tables**.
  * **Privacy-First:** Secure data with local storage and optional Google Drive/OneDrive backups.
